To implement incremental retrieval, enable one of the following options:

  • Incremental Retrieval Since - Adds the condition SystemModstamp > (Last Start Time) to the Retrieve Parameters during execution. The Last Start Time refers to the Start Time of the most recent SUCCEEDED Execution.
  • Incremental Retrieval Seed Time - Adds the condition SystemModstamp > (Last End Time) to the Retrieve Parameters during execution. The Last End Time refers to the End Time of the most recent SUCCEEDED Execution.

Note: When updating the source query, be sure to Empty icremental retrieval first. Re-enable it after the query has been updated to ensure proper evaluation of changes.
