Navigate to the Connection record for the remote Salesforce org,

go to the Q tab, and execute your query using Query Manager(Q).

  • The data retrieved reflects the access/permissions of the authorized user defined in the Connection.
  • To access the Q tab on Connections, users must have the custom permission - DSP: Access Q on Remote Connections.

This permission is included in DSP: Data Sync Pro Administrator. Otherwise, assign it manually as needed.
