Mapping in Data Sync Pro (DSP) defines how values are assigned to target fields before execution. Through an intuitive, no-code interface, users can apply powerful, formula-based transformations to clean, enrich, and structure data at scale—ensuring accuracy and consistency across operations.

Key Capabilities:

  • Powerful & Bulkified Transformation
    Apply bulkified formulas using DSP's rich function library with support for referencing source fields (including relational fields), static values, Variables (reusable formula definitions), and optional custom Apex classes.
  • Relationship-Aware Assignments
    Easily handle data relationships with support for Record Type mapping via Developer Name, External ID references, and VLOOKUP-based lookups.
  • Autocomplete & Validation
    Benefit from real-time autocomplete hints and formula validation on save, improving accuracy and ease of use.
  • Single-Record Simplicity, Bulk Execution
    Define mappings as if configuring a single record— DSP automatically applies them at scale, handling large volumes with optimized, bulkified execution.

Mapping provides a scalable, low-code transformation layer that balances power, flexibility, and simplicity—enabling precise control over how data is prepared before it reaches its destination.
