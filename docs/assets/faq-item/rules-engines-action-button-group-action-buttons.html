Grouping Executable Action Buttons into a Pipeline offers several key benefits:

  1. Centralized & Organized UI
    The Pipeline Action Buttons component can be placed on a Lightning page to present multiple buttons in a unified, streamlined layout—making it easier for users to access related actions from one place.
  2. Flexible Configuration & Maintenance
    You can add, remove, reorder, or modify Executable Action Buttons directly within the Pipeline—without changing the Lightning page—making updates easier to manage.
  3. Streamlined User Experience
    Displaying related actions together reduces UI clutter and improves usability by aligning buttons with business processes in a clean, consistent interface.

Pipelines bring structure, flexibility, and modularity—making it easier to manage and present multiple related #Action Buttons in one place.
