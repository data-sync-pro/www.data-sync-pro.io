How does a dynamic filter work?

A dynamic filter in Data Sync Pro allows query filters to adjust automatically based on a record context or the current user, using custom Apex code (low-code) where needed. This enables context-aware queries that dynamically respond to field values from a specified record, adapt to the attributes of the logged-in user, or execute fully customized logic.
