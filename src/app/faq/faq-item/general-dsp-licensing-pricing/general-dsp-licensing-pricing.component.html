Data Sync Pro uses a site-wide license model, meaning it can be assigned to unlimited users within your Salesforce org. This allows you to maximize value across teams without incurring per-user (per-seat) costs.

Pricing is based on three key factors:

  • Number of Connections – The number of external Salesforce orgs or environments connected through DSP.
  • Number of Executables – The number of rules or logic units created in DSP to define data operations.
  • Daily Data Volume – The total amount of data DSP processes each day.

This model ensures you're paying for the value delivered—based on the scale and complexity of your data management needs—not the number of users accessing the app.
