The "Refresh" button reloads the list of available source and target fields in the Mapping section, ensuring that any newly added fields in the source or target objects are reflected and available for mapping.

Additionally, DSP re-sorts the target fields into the following categories for better organization:
1. Target Matching Field
2. Mapped Fields with Seq No.
3. Reference Fields
4. Non-Reference Fields

Within each category:
- Mapped fields appear before non-mapped fields.
- Target fields are sorted alphabetically.

This structured sorting enhances usability and simplifies field selection.
