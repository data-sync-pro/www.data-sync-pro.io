When you run a query in Query Manager (Q), you get an interactive table with tools to explore, edit, and manage data—along with a breakdown of your SOQL query for easy adjustments.

Key Features on Execution:

  • Query Decomposer – Automatically parses the query into intuitive UI components for each clause—SELECT, WHERE, ORDER BY, and GROUP BY—making it easy to understand and modify.
  • Results Display – Shows retrieved records, record count, and object label, with pagination controls.
  • Column Filters – Filter results hinted based on field type: picklist, date, number, text, or reference.
  • Sorting & Labels – Displays field labels and allows click-to-sort by any sortable column.
  • Inline Editing – Edit supported fields like picklists, references, and text directly in the table.
  • Row Actions – Each record supports View, Edit, Clone, and Delete options.
  • New Record – Add new records quickly via the built-in button.
  • Mass Edit – Update multiple records in bulk.
  • Mass Blank Update – Trigger automations without changing values.
  • Mass Delete – Delete selected records in one action.
  • Export – Download full or selected results as CSV or JSON.
  • Picklist Toggle – Switch picklist fields between API values and user-friendly labels.
  • Record Navigation – Click ID, Name, or reference fields to open either the DSP raw record view or the standard Lightning record page, depending on your setting.
