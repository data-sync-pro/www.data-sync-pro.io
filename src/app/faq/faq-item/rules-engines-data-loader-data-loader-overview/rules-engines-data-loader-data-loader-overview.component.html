Data Loader in DSP is a record-based rules engine designed for processing CSV file uploads, applying transformations, and executing actions in Salesforce. Unlike other DSP rules engines that retrieve data from a Salesforce object, Data Loader processes records directly from a CSV file and performs operations like Insert, Update, Upsert, Delete, and more based on predefined logic.
