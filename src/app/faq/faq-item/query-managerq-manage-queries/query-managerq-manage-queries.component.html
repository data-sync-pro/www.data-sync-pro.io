In Query Manager (Q), you can efficiently save, organize, and reuse your queries:

  • Save Queries – Use "Save Query" to store the current query for future use.
  • View Saved Queries – Click "Select Query" to browse your saved queries, sorted by Last Modified Date.
  • Select a Query – Load any saved query, modify it, and execute to view updated results.
  • Sort & Filter – Sort by Label or Object API Name, and use column filters for quick navigation.
  • Relabel or Delete – Update query labels or remove unneeded queries.
