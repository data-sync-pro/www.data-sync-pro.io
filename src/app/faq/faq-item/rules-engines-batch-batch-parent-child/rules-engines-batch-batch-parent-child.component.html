Go to Migration Builder within the Pipeline to set up Batch Executables for syncing multiple objects.

  1. Create a Pipeline record and set its direction to Template.
  2. Navigate to the Migration Builder tab.
  3. Add root objects, then expand each root object down to the leaf node you want to sync.
  4. Select the desired leaf node.
  5. In the modal popup, choose the source and target matching fields, define the action, and click Create Executables to generate the required batch processes.
