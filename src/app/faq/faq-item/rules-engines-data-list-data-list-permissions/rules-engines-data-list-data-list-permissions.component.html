To access and use a Data List (including Actionable Data Lists), users need the following:

  1. 1️⃣ DSP Permission Set
    1. - Minimum required: DSP: Data Sync Pro Starter
  2. 2️⃣ Executable Record Access
    1. - Must have at least Read access to the Executable record that defines the Data List.
  3. 3️⃣ Custom Permission (View Access)
    1. - If “Q: Required Custom Permissions To View” is set, users must have the specified custom permission(s).
  4. 4️⃣ Custom Permission (Action Access)
    1. - For Actionable Data Lists, if “Required Custom Permissions To Execute” is configured, users must have those permission(s) to see and use action buttons.
  5. 5️⃣ Object & Field-Level Access
    1. - Users must have Read access to the source object and fields used in the query.
