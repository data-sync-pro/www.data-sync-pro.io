A Scope Filter in Data Sync Pro (DSP) is a formula-based condition used during the Scoping step to determine which source records should proceed through execution. Scope Filters can either:

  • Evaluate to TRUE or FALSE to include or exclude records, or
  • Raise a conditional error using the ERROR() function when validation criteria are met.

Scope Filters are evaluated sequentially, allowing for layered, modular filtering logic based on context:
