When maintaining data relationships for reference fields in DSP:

  • If you know the record ID, assign it directly.
  • For Record Types, you can assign the Developer Name instead of the ID.
  • For referenced objects with External ID fields, select the Referenced Object and the corresponding External ID field, then map the External ID value in the field's mapping.
  • Alternatively, use the VLOOKUP function to find the record ID based on field matching. DSP supports multiple field matching within VLOOKUP functions for enhanced flexibility.
