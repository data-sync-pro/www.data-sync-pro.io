DSP Data Loader supports transformations by applying formulas and expressions in Scoping and Field Mappings. To enable this, column names in the CSV file must follow a strict format:

  • Only English letters, numbers, and underscores are allowed.
  • The column must start with a letter.
  • The column cannot end with an underscore.
