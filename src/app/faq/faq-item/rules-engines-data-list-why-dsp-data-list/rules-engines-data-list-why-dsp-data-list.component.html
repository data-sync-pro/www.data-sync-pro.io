The DSP Data List is a powerful, no-code, record-based rules engine built to let Salesforce users retrieve, transform, and take action on Salesforce data in real time—all through a guided, configurable interface. With support for inline editing, dynamic filters, row and list actions, and seamless Lightning integration, it provides a highly interactive and user-friendly experience. Its modular design, declarative setup, and low-code maintenance model empower teams to deliver flexible, context-aware data solutions—while minimizing development effort and reducing tech debt.


Key Advantages of DSP Data Lists:

  • Fast & Flexible Setup – Define a single SOQL query using the Query Builder (Q) and instantly turn it into a live, interactive data table.
  • Robust User Interactions – Empower users with:
    • Inline editing, mass editing, record creation, cloning, and deletion
    • CSV upload and CSV export
    • Customizable related lists to show contextually linked records
    • Column filters for quick, user-friendly field-level filtering
    • Automatic pagination for seamless navigation across large datasets
  • Powerful Data Transformations – Apply formula-based logic to dynamically manipulate and display data, enhancing clarity and enforcing consistency.
  • Custom Row & List Actions – Trigger record updates, lead conversions, email sends, notifications, or multi-step logic from within the list.
  • Advanced Filtering Options – Use dynamic filters driven by user context or record parameters for intelligent, personalized data access.
  • Streamlined Visual Customization – Configure layout, visibility, and behavior through an intuitive, declarative interface—without writing code.
  • Modular & Scalable Design – Group multiple Data Lists into Pipelines to create guided, multi-step workflows that remain maintainable and reusable.
  • Cross-Org Data Interaction – Retrieve and act on data across connected Salesforce orgs using Connection-based Executables, all within DSP.
  • Seamless Lightning Integration – Embed in Lightning Pages using tabs, stacks, or list-style views for a unified, interactive experience.
  • Enterprise-Grade Access Control – Control who can see and interact with each Data List and its defined actions based on:
    • Permission sets
    • Custom permissions
    • Executable sharing
    • Target object access
    This ensures only the right users can view, edit, or take actions—honoring both DSP-specific and Salesforce-native security.

As an intuitive, no-code engine built for admins, developers, and business users alike, DSP Data Lists provide unmatched control and automation for interacting with Salesforce data—at scale, across orgs.
