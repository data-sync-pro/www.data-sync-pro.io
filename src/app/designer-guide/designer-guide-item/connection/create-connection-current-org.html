Connection

A Connection (pushtopics__Connection__c) defines authorization settings that can be used to connect to a Salesforce instance, either as a source (data retrieval) or a target (data synchronization).

Fields

Name API Name Required Description
API Name pushtopics__ApiName__c Y The unique, external-ID field for a Connection. Hidden from page layout by default and defaulted to the Name field value.
Connection Name Name N The human-readable name of the Connection.
Is Current Org? pushtopics__IsCurrentOrg__c N If checked, executes as the current user via Apex (no external credentials).
Is Sandbox? pushtopics__IsSandbox__c N Together with Username and Password + Security Token, indicates a sandbox environment.
Named Credential pushtopics__NamedCredential__c N Developer name of the Named Credential. Required settings:
  • Identity Type: Named Principal
  • Auth Protocol: OAuth 2.0 or Password Authentication
  • Allow Merge Fields in HTTP Header & Body
Use https://login.salesforce.com for production, https://test.salesforce.com for sandbox.
Password + Security Token pushtopics__SalesforcePassword__c N Combined password and token for API authentication (with Username and Is Sandbox?).
Username pushtopics__Username__c N The Salesforce username used to authenticate.
