To grant users access to a #Data Loader:

  • Assign the "DSP: Data Sync Pro App User" permission set, which provides the necessary access to execute a #Batch or #Data Loader, including create access to Execution-related objects. Additionally, ensure the Executable record defining the #Data Loader is shared with the user through record sharing or sharing rules.
  • Alternatively, assign the "DSP: Data Sync Pro Administrator " permission set, which grants full access to all DSP functionalities, including #Data Loaders.
