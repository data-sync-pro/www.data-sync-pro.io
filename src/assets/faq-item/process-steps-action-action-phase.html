The Action step in Data Sync Pro (DSP) defines how processed data is applied to the target object. It comes after Scoping, Match, and Mapping, and determines what operation is performed on each target record.

Key Capabilities of Action:

  • Perform Target Operations
    Supports standard DML actions like Insert, Update, Upsert, Delete, Undelete, Merge, as well as extended operations such as Lead Conversion, Send Email, Bell Notification, etc.
  • Execution Behavior Controls
    Fine-tune how each record is processed using configurable options:
    • Skip Record Update If No Changes
    • Skip Fields If Target Value Exists
    • Skip Null Value Fields
    • Bypass Duplicate Rule Alerts
    • All or Nothing DML Execution
  • Writeback Options
    Allows writing specific values back to the source or logging them for reference after successful execution.
