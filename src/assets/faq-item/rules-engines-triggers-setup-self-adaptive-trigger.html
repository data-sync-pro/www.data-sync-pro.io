On a "Current => Current" Direction record or a Pipeline with a Template Direction set to "Current => Current", click New Executable.

Select the same object as both the source and target, use the "Id" field for matching, and choose either "Update" or "Upsert" as the action. Disable "Batchable", then create the Executable.


Once created, navigate to the Triggers tab to configure the detailed logic for the Self-Adaptive Trigger.
