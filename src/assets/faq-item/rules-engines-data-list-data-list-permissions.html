To access and use a #Data List (including Actionable Data List), users need the following:

  1. DSP Permission Set
    • Minimum required: DSP: Data Sync Pro Starter
  2. Executable Record Access
    • Must have at least Read access to the Executable record that defines the #Data List.
  3. Custom Permission (View Access)
    • If "Q: Required Custom Permissions To View" is set, users must have the specified custom permission(s).
  4. Custom Permission (Action Access)
    • For an Actionable Data List, if "Required Custom Permissions To Execute" is configured, users must have one of those permission(s) to see and use the action.
  5. Object & Field-Level Access
    • Users must have necessary access to the source and target data involved in the #Data List.
