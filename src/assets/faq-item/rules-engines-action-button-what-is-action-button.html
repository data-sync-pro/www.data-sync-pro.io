An Action Button in Data Sync Pro (DSP) is a no-code, record-based rules engine that enables business users to trigger data transformations and actions instantly with a single click—directly from the Salesforce Lightning UI.

Each Action Button is defined in an Executable record, with a guided user interface that makes configuration intuitive and maintainable. It supports powerful transformations using DSP's formula engine and allows users to review and edit transformed field values before execution.

Supported actions include Insert, Update, Delete, Merge, Lead Conversion, Send Emails, Publish Platform Events, Bell Notifications, and more.

Action Buttons can be added to Lightning pages as Quick Actions or surfaced as a customizable group of buttons using a Lightning Web Component—empowering business users with real-time, rules-driven automation inside Salesforce.
