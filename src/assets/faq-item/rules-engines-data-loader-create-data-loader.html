To create a Data Loader in Data Sync Pro (DSP), follow these steps:

  1. Go to the Connection Record
    • Navigate to the Connection where you want to upload data. If loading into your current Salesforce org, use the Current Org connection.
  2. Create a New Executable
    • Click “New Executable (Action)”
    • Set the Type to Data Loader
    • Specify the Target Object, choose an Action (Insert, Update, Upsert, Delete), and define a Target Matching Field if needed
  3. Upload a Template CSV File
    • Upload a CSV file with headers—DSP uses this to generate the data profile
    • Make sure column names follow DSP naming conventions: use only letters, numbers, and underscores, and start with a letter
  4. Configure the Data Loader Steps
    • Once created, the Executable will guide you through the following steps:
      • Input - Define the structure of the uploaded CSV so its fields can be used in filters and field mappings.
      • Scoping - Refine the retrieved dataset using in-memory filters and optional joins with other datasets for multi-source enrichment.
      • Match - Identify target records using matching rules. Supports insert-only logic, multi-field matching via transformations, and merge-specific logic.
      • Mapping - Apply formulas and expressions to assign and transform target field values. Features auto-validation, relational field support, and a comprehensive function library.
      • Action - Define the operations to be performed, such as Insert, Update, Upsert, Delete, Merge, Lead Convert, Send Email, and more. Configure behavior options like “Skip Record Update if No Changes”

Once configured, your Data Loader can be reused to process similar datasets—ensuring fast, accurate, and consistent uploads across your org.
