Source Object Writeback updates the source record after its related target records are successfully actioned. This helps with tracking, marking completion, and preventing duplicate processing.

When a Source Writeback Field is configured, DSP supports the following options:

  1. Target Record ID(s) – Writes a comma-separated list of successfully processed target record IDs back to the source record.
  2. Static Value (TRUE or FALSE) – Writes either TRUE or FALSE to the specified source field.
