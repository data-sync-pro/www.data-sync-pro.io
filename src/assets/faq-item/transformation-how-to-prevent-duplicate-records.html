Use the IS_FIRST_IN_ITERATION(...expressions) function in a Scope Filter. It returns TRUE the first time a given combination of evaluated expression values occurs in the current iteration, and FALSE for any duplicates after.

Example:

IS_FIRST_IN_ITERATION(FirstName, LastName)  
-- TRUE for the first FirstName + LastName pair, FALSE for subsequent duplicates

Since the parameters can be any DSP expressions (not just fields), you can deduplicate on calculated values, formulas, or combined logic evaluated at runtime.
