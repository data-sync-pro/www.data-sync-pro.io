Enabling Bulk API in Data Sync Pro (DSP) delivers major performance benefits but also introduces important trade-offs:

  • Benefits

    • Extraordinary Performance Gains – With larger batch sizes (up to 2,000 records per batch), Bulk API can process large data volumes significantly faster than the standard API.

  • Drawbacks

    • Asynchronous Processing – Bulk API jobs run asynchronously after DSP’s batch submissions. You must monitor execution status directly in Salesforce’s Bulk Data Load Jobs.

    • No Data Reversion – Actions performed via Bulk API cannot be reverted. Features such as Restore Updated From Log File or Re-Create Deleted From Log File are not supported.

Bulk API is ideal for high-volume processing, but it requires careful setup and active monitoring for optimal use.
