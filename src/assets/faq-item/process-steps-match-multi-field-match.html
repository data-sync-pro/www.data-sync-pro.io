To match target records using multiple fields, use a `VLOOKUP` formula in the Target Matching Field. This allows you to locate the correct record based on multiple criteria.

Example: VLOOKUP("PermissionSetAssignment", "Id", "AssigneeId", Id, "PermissionSet.Name", "DataSyncProStarter")
  • Target Matching Field: Id (of PermissionSetAssignment)
  • Source Object: e.g., User, using the Id field
  • Matches on: AssigneeId (User Id) and PermissionSet.Name

If multiple matches are found, use the Principal Matched Record Selection Rule to control which record is selected.
