Query Manager(Q) is a Lightning Web Component (LWC) in Data Sync Pro that enhances how users build, execute, and interact with SOQL queries in Salesforce. Designed for architects, admins, developers, and business users alike, it provides a structured, interactive interface that enables fast, accurate, and consistent data exploration and management across orgs.

Key Capabilities

  • SOQL Builder - Build queries with ease using a guided interface and smart autocomplete, minimizing errors and simplifying logic creation.
  • Advanced Data Interaction - Access inline editing, mass update/delete, record creation, cloning, pagination, column filtering, and data exportâ€”directly from the query results.
  • Manage Reusable Queries - Save, organize, and reuse queries for consistent workflows and efficient collaboration.
  • Query Decomposer - Automatically breaks down raw SOQL into structured parts like SELECT, WHERE, ORDER BY, and GROUP BY, making complex queries easier to understand and adjust.
  • Dynamic Filters - Apply user- and record-based filters to generate context-aware, personalized data views.
  • Seamless Lightning Integration - Q works with DSP Executables to define powerful Actionable Data Lists that can be embedded into Lightning Pages for business operations and real-time interaction.
