Mapping defines how target fields are calculated before execution. It acts as the transformation layer that drives data accuracy, consistency, and quality. With a guided interface and extensive function library, designers can configure powerful transformations with ease.

Key Capabilities:

  • Formula-Based Transformations: Apply formulas from DSP’s rich function library, referencing source fields (including relational fields), static values, Variables (reusable formulas), and optional custom Apex classes.

  • Data Relationships: Seamlessly handle External IDs, VLOOKUPs, and Record Type names.

  • Autocomplete & Validation: Get real-time hints and formula validation on save for accuracy.

  • Automatic bulkification: Define mappings as if for one record—DSP automatically bulkifies the logic for optimized performance.
