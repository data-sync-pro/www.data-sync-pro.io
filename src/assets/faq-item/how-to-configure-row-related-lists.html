To display related lists for each row in a #Data List:

  1. Create a Pipeline containing one or more #Data Lists, each configured with dynamic filters based on a Context Record ID.
  2. In the parent Data List Executable, set "Row's Related Lists Pipeline API Name" to the API Name of the Pipeline created in Step 1.
Once configured, a standard "Related Lists" row action automatically appears for each row. Clicking it displays the related lists defined in the Pipeline.
  • By default, related lists open in a modal popup.
  • To show them inline beneath the main Data List, set "Show Row's Related Lists Below?" to TRUE.
