An Actionable #Data List is created as an Executable associated with a Direction. You can create one by clicking New Executable from:

  • A Direction record, or

  • A Pipeline with a Template Direction specified

Once created, DSP provides a guided interface (Retrieve, Scope, Match, Mapping, Action, Verify, and additional Data List (Q) settings) to configure the Data List into a fully interactive, reusable datatable.
