In Query Manager(Q), you can efficiently save, organize, and reuse your queries with the following features:

  • Save Queries - Use the "Save Query" button to store your current query for future use.
  • View Saved Queries - Click "Select Query" to browse saved queries, sorted by Last Modified Date.
  • Select a Query - Load any saved query into Q, modify it as needed, and execute the query to view the results.
  • Sort & Filter - Sort by Label or Object API Name, and use Column Filters to find queries quickly.
  • Relabel or Delete - Update query labels for clarity or remove unused queries.
