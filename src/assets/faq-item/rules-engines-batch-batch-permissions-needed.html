While executing a #Batch job in DSP, the running user's access determines what data they can see and act on. However, to execute a #Batch job in DSP, users need:

✅ Read access to the Batch Executable

✅ Create access on Execution log related objects

Permission Sets:

Permission Set Batch Access
DSP: Data Sync Pro Administrator ✅ Full access; can run any Batch
DSP: Data Sync Pro App User ✅ Can run shared Executables (requires record sharing)
DSP: Data Sync Pro Starter ❌ No execution access; cannot run Batches even if Executables are shared
