When multiple Executable Data Lists in a Pipeline share the same source query, their actions are automatically merged into one unified Data List. This lets users work with a single list while still accessing all actions defined across those Executables.

To enable this:

  • Ensure the Executables in the Pipeline use the same source query.

  • Add the Pipeline Data Lists component to a Lightning page.

  • Configure the Data Lists within the Pipeline to control order of the actions

This approach streamlines the user experience by eliminating duplication and consolidating actions into a single, interactive view.
