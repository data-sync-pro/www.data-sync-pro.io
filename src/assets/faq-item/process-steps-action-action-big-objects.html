When the target object is a Salesforce Big Object, DSP supports the following actions:

  • Insert - Adds new records. If a record with the specified index fields already exists, Salesforce will override the existing record, even though the action is set to "Insert." This is a standard Salesforce functionality.
  • Delete - Removes existing records based on the index.

"Target Big Object Index Fields" must be configured to ensure proper data processing.
