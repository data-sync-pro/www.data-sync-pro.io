Data Sync Pro (DSP) includes five distinct rules engines—each purpose-built for a different type of data operation—yet all sharing a common record-based configuration model.

#Batch
Optimized for large-volume processing. Replaces Apex batch jobs, supports complex transformations, data cleansing, and high-performance migrations between Salesforce orgs.

#Data List
Turns query results into interactive, configurable tables. Business users can filter, explore, edit, and act on records with list-level or row-level interactions.

#Triggers
Runs automation in real time based on DML events (insert, update, delete, undelete). Ideal for immediate transformations, validations, and actions.

#Action Button
Adds configurable buttons in the Lightning UI for guided record-level or global actions. Supports pre-populated transformations, validations, and updates.

#Data Loader
Automates CSV uploads with reusable logic for transformation and validation. Provides secure, trackable imports into any connected Salesforce org.

Together, these engines form DSP’s unified automation platform—giving teams modular, flexible, and enterprise-grade data processes, all without code.
