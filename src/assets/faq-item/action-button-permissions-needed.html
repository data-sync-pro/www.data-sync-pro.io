To access and use a #Action Button, users need the following:

  1. DSP Permission Set
    • Minimum required: DSP: Data Sync Pro Starter
  2. Executable Record Access
    • Must have at least Read access to the Executable record that defines the #Action Button.
  3. Custom Permission (Action Access)
    • If “Execute Access Permissions” is configured, users must have one of those permission(s) to see and use the action.
  4. Object & Field-Level Access
    • Users must have necessary access to the source and target data involved in the #Action Button.
